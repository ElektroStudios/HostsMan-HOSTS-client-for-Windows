## 👋 Introduction

This project is a simple, lightweight Windows desktop application built with WinForms and VB.NET. It provides a straightforward graphical user interface (GUI) to view, edit, and organize your Windows HOSTS file without having to manually open text editors and the risk of corrupting its structure.

## 💡 Motivation

Managing the Windows HOSTS file for local domain mapping, development testing, or blocking unwanted domains is usually a clunky, text-based task. Opening Notepad as Administrator, navigating to deep system folders (`System32\drivers\etc`), and dealing with accidental formatting issues can be tedious. I wanted to build a humble, reliable utility that makes this workflow quick and secure through a clean, native interface.

##### ⚡ The Real Question
###### Why waste time digging through Windows system directories and editing raw text files when you can safely manage, backup, and organize your host mappings with just a few clicks?.

## 🚀 Getting Started

1. Navigate to the **[Releases page](https://github.com/ElektroStudios/HostsMan-HOSTS-client-for-Windows/releases/latest)**.
2. Download the latest `.zip` archive or the `.exe` setup installer, depending on your preference.
3. If you downloaded the `.zip` archive, extract its contents to your preferred directory.
   <br> 
   If you downloaded the `.exe` file, run it and follow the installation wizard.
4. Run the executable file to launch the application.
